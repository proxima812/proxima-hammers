# Proxima Hammers

Fabric-only Minecraft mod that adds a progression of hammers and upgrade matrices.

## Features

- Wide-area mining hammers from stone through netherite, plus custom material tiers.
- Improved hammer variants with stronger stats and larger mining depth.
- Crafting-table matrix upgrades for haste, magnetism, knowledge, power, and Thor flight.
- Thor Matrix flight with a double-jump launch, high-speed glide, white magic particles, and impact/fall protection while holding the hammer.
- Datagen-backed recipes, item models, tags, and translations.

## Requirements

- Minecraft `26.1` through `26.1.2` for this build
- Fabric Loader `0.18.4` or newer
- Fabric API `0.149.0+26.1.2` or newer
- Java `25`

The current build compiles against Minecraft `26.1.2`. Older Minecraft versions are released from dedicated `mc/*` compatibility branches.

## Build

```powershell
.\gradlew.bat build
```

The release jar is generated at:

```text
fabric/build/libs/proxima-hammers-fabric-1.0.jar
```
