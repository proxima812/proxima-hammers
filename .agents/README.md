# Agent Tools

This directory stores local automation and project guidance for Proxima Hammers.

- `hooks/preflight.ps1` checks project identity, forbidden legacy names, resource namespace consistency, and Gradle build health.
- `skills/proxima-hammers/SKILL.md` describes how to modify this mod without losing existing modules, textures, recipes, or translations.
- `settings.json` records the expected identity and deployment policy for lightweight tooling.
